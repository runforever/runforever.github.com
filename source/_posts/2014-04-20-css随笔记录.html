CSS 选择器优先级

!important > [ id > class > tag ] 使用!important可以改变优先级别为最高，其次是style对象， 然后是id > class >tag ，另外，另外在同级样式按照申明的顺序后出现的样式具有高优先级。

css important 语法 

div {
    width: 40px!important;
}
